# MiliUI BloodlustMusic

A World of Warcraft addon that plays music and shows a countdown bar whenever Bloodlust / Heroism / Time Warp / Fury of the Aspects / Primal Rage (or Leatherworking Drums) is active.

![logo](logo.png)

## Features

- Plays a 40-second music track when a Lust buff is detected
- Movable countdown bar showing remaining Lust duration
- Reminder popup to cast Lust (dungeon pull / debuff expiry)
- Supports built-in tracks and custom user-added tracks (MP3 / OGG)
- Random or sequential play mode
- Per-class Lust reminder filtering
- Localization: English, 繁體中文, 简体中文, 한국어

## Installation

1. Download the latest release ZIP
2. Extract to `World of Warcraft\_retail_\Interface\AddOns\`
3. Restart the game

## Usage

- Type `/blm` in chat, or
- `Esc` → `Options` → `AddOns` → `Bloodlust Music`

## Adding Custom Music

1. Drop your `.mp3` or `.ogg` file into the addon's `Media\` folder
2. Open the options panel and enter the filename under Custom Tracks

# MiliUI 嗜血音樂

一個《魔獸世界》插件，當偵測到嗜血／英勇／時空扭曲／巨龍怒火／原始之怒（或製皮戰鼓）時，自動播放音樂並顯示倒數計時條。

![logo](logo.png)

## 功能

- 偵測到嗜血效果時，播放 40 秒音樂
- 可拖曳的倒數條，顯示嗜血剩餘時間
- 嗜血施放提醒（副本開怪 / 虛弱 debuff 即將結束）
- 支援內建曲目與玩家自訂曲目（MP3 / OGG）
- 隨機或順序播放模式
- 依職業過濾嗜血提醒
- 多語系：English、繁體中文、简体中文、한국어

## 安裝

1. 下載最新版本的 ZIP 檔案
2. 解壓縮至 `World of Warcraft\_retail_\Interface\AddOns\`
3. 重新啟動遊戲

## 使用方式

- 聊天輸入 `/blm`，或
- `Esc` → `選項` → `插件` → `嗜血音樂`

## 新增自訂音樂

1. 將 `.mp3` 或 `.ogg` 檔案放入插件的 `Media\` 資料夾
2. 開啟設定面板，在「自訂曲目」中輸入檔名
